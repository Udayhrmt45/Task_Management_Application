# Task Management System

A React application built with Vite for managing tasks and tracking progress. It includes user login, task search and filters, and paginated task listings.

## Setup

1. Clone the Repository

```bash
git clone https://github.com/Udayhrmt45/Task_Management_Application.git
cd Task_Management_System
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the application in your browser at the address shown by Vite, typically:

```bash
http://localhost:5173
```

## Features

- User login flow with access control for the dashboard.
- Task list with search and status filter options to quickly find tasks by title or assigned user.
- Paginated task display so users can browse large task lists one page at a time.
- Task cards that show key details such as title, assignee, status, and description.
- Add task page for creating new tasks with form validation and immediate list updates.
- Responsive layout that adapts for desktop and mobile screens, keeping the interface usable across devices.
- Real-time task status filtering, letting users view tasks marked as In Progress, Completed, or On Hold.
- Clean dashboard layout with summary stats and a search bar for efficient task management.
